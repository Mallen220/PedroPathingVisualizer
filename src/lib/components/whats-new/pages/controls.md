# Controls and Shortcuts

Here are the controls and shortcuts to help you navigate the application efficiently.

## General

- **Ctrl + S**: Save Project
- **Ctrl + O**: Open Project
- **Ctrl + Z**: Undo
- **Ctrl + Shift + Z**: Redo

## Canvas

- **Scroll**: Zoom in/out
- **Middle Click + Drag**: Pan the view
- **Left Click**: Select point
- **Right Click**: Context menu

## Path Editing

- **Delete**: Remove selected point
- **Esc**: Cancel current action

# Controls and Shortcuts

Here are the controls and shortcuts to help you navigate the application efficiently.

## General

- **Ctrl + S**: Save Project
- **Ctrl + O**: Open Project
- **Ctrl + Z**: Undo
- **Ctrl + Shift + Z**: Redo
- **Shift + D**: Duplicate Selected Path

## Canvas

- **Scroll**: Zoom in/out of the field view.
- **Middle Click + Drag / Space + Drag**: Pan the view.
- **Left Click**: Select point
- **Right Click**: Context menu

## Path Editing

- **Delete**: Remove selected point
- **Esc**: Cancel current action

## Customizing Shortcuts

You can view and customize all keyboard shortcuts in the **Settings** menu under the "Keyboard Shortcuts" section.

### What's New!

## **Features:**

- **PedroPathingPlus**
  - Added native support and documentation for PedroPathingPlus, an advanced command-based pathing library.
  - Learn how to integrate direct `.pp` execution and advanced features into your auto.

- **Lock Obstacles**
  - You can now lock individual obstacles to prevent accidental changes.
  - Locked obstacles cannot be moved, resized, or deleted until unlocked.

- **Autosave is Here!**
  - Your work is now saved automatically as you make changes. No more panic over lost progress.
  - Safe file switching ensures your data is secure before moving to another project.

- **Smart Onion Skins**
  - Added a new setting to show onion skins only for the current path segment during animation.
  - This reduces clutter and helps you focus on the immediate motion of the robot.

- **Drag-and-Drop File Support**
  - Easily load `.pp` files by dragging and dropping them into the app.
  - Streamlined workflow for working with multiple project files.

- **Streamlined Setup**
  - A new setup dialog guides you to select your project directory when you first launch the app.
  - No more automatic folder creation in unexpected places.

- **Angular Velocity & Acceleration**
  - Angular Velocity settings now correctly affect path timing calculations.
  - New Angular Acceleration setting for more realistic robot motion profiles.
  - Toggle between Deg/s and other units for angular measurements.

- **Enhanced Robot Profile Manager**
  - Improved UX for managing robot profiles with better organization and accessibility.

- **Event Management**
  - Replaced text input for events with a searchable dropdown for faster, more intuitive selection.
  - Implemented to try to decrease number of incorrectly named events.

## **Improvements:**

- **Better Dialog Handling**
  - The "Unsaved Changes" dialog now correctly closes when pressing `Escape`.
  - Improved keyboard accessibility for dialogs.
  - Enhanced close and save checking logic for better data safety.

- **Better Keyboard Shortcuts**
  - `Cmd+R` (or `Ctrl+R`) now creates a **New Project** instead of reloading the window.
  - Disabled disruptive shortcuts like standard reload and window close to prevent accidental data loss.
  - Disabled Chromium default keyboard shortcuts for a safer, distraction-free experience.
  - Always more keybinds for power users and improved workflow efficiency.

- **Fixed Bugs**
  - Corrected rotation logic for linked points to ensure accurate transformations.
  - Fixed faulty Linux support for better cross-platform compatibility.
  - Improved new project reset logic to prevent state inconsistencies.

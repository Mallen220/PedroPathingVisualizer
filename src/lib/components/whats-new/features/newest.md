### What's New!

## **Features:**

### **Enhanced Path Analytics**
The **Path Statistics** dialog has been upgraded to provide deeper insights into your robot's performance:
*   **New Acceleration Graphs:** Visualize both linear and centripetal acceleration to identify potential stability issues.
*   **Safety Insights:** A new "Insights" tab warns you about potential problems:
    *   **Wheel Slip:** Detects if the path requires more friction than available.
    *   **Max Velocity:** Flags sections where the robot exceeds safe speed limits.
*   **Consolidated Warnings:** Similar issues are grouped by time range for easier reading.

### **Comprehensive Keyboard Navigation**
Full keyboard support is now available for managing paths and sequences:
*   **Sequence Navigation:** Use `Tab` / `Shift+Tab` to move between high-level sequence items (Paths, Waits, Rotations).
*   **Smart Scrolling:** The sidebar now automatically scrolls to keep the selected item in view.
*   **Edit Mode:** Press `Enter` to immediately edit the properties of the selected item.
*   **Quick Deselect:** Press `Escape` to blur input fields or deselect items.
*   **Obstacles Access:** Obstacles are now visible and editable directly from the Table tab.

### **Accessible Color Picker**
*   The color picker in the Palette now supports keyboard focus, making it fully accessible for all users.

## **Bug Fixes:**
*   **Path Statistics:** Fixed an issue where warning insights could display incorrect end times.
*   **General:** Updated Discord server invite links.

### What's New!

## **Features:**

### **Improved Accessibility**

We've enhanced the keyboard navigation and visual feedback for the **Playback Controls**.

*   **Keyboard Support:** You can now fully control the timeline slider, markers, and buttons using your keyboard.
*   **Focus Indicators:** Clear focus rings have been added to all interactive elements, making it easier to see what is selected.

### **Debug Sequence Visualization**

A new tool for advanced users to better understand path execution.

*   **Visualize Order:** View the internal execution order of your path sequences directly on the field.
*   **Enable:** Toggle "Show Debug Sequence" in the **Advanced** tab of the Settings dialog.

## **Bug Fixes:**

*   **Release Counter:** Fixed an issue where the download count for releases might display incorrectly or fail to load.

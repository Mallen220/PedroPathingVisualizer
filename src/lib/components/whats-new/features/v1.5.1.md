### What's New in 1.5.1

We've been hard at work! This update includes critical fixes and a host of new features introduced in version 1.5.0.

**Latest Changes (1.5.1):**

- **Context Menu Updates**: The waypoint table context menu now has more options for easier control.
- **Critical Fixes**: Addressed a critical error preventing startup on some Windows systems.
- **New "What's New" Dialog**: Introduced this dialog to keep you informed about updates directly within the app.

**Major 1.5.0 Features:**

- **UI Overhaul**:
    - **Navbar**: Redesigned for better accessibility and aesthetics.
    - **Keybinds**: A completely new interface for viewing and customizing keyboard shortcuts.
    - **File Manager**: Improved navigation and reliability.
- **New Tools**:
    - **Duplicate Path**: Quickly duplicate a path using `Shift + D`.
    - **Reverse Path**: Easily create a path that follows the reverse trajectory of an existing one.
    - **Field Validation**: New settings to enforce field boundaries and restrict dragging outside valid areas.
- **Exporting**:
    - **APNG Support**: You can now export your path animations as Animated PNGs, offering better quality than GIFs.
- **Optimization**:
    - **Mutation Strength**: Increased default mutation strength for the path optimizer, leading to faster and better convergence.
- **Field View**:
    - **Zoom & Pan**: Added intuitive zoom controls and panning to the field view.
    - **Performance**: Optimized rendering logic for smoother interaction.

Version 2.0.0 Highlights

Whats New in 2.0.0

Major Changes:

  • Rewrite of core pathfinding engine with a plugin architecture.
  • Introduced first-party cloud sync for project backups (opt-in).
  • New theming engine and accessibility improvements (contrast and keyboard nav).

Features:

  • Plugin system: developers can write pathfinding or visualization plugins.
  • Online galleries to share field presets and community templates.

Breaking Changes:

  • Project file format updated older projects will be migrated on first open.
  • Removed deprecated legacy-path export option use export-v2 instead.

Migration Notes:

  • A one-time migration dialog will guide you through any required conversions.

Thanks:

  • Huge thanks to everyone who tested the beta and provided feedback.
